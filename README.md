Dual Self-Expression Subspace Clustering with Multi-Scale Features and Structural Consistency Alignment

This repository contains the implementation of a deep clustering framework
based on multi-scale visual features, graph-based structural learning, and
dual self-expression with consistency alignment.

USE

step 1. Clone code and prepare environment.

pip install -r requirements.txt

step 2.feature_extract.
python Feature_extract.py --dataset=fashion_mnist

step 3.Training 
python main.py --dataset=fashion_mnist
